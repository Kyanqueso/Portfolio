Projects

Toy Tracker

The Toy Tracker App (a personal project) is an offline toy tracking application that helps users manage their toy collection with ease. It features a clean, user-friendly interface for adding, viewing, and updating toy details, including images, costs, suppliers, and payment status. With built-in search, filtering, and detailed overlays, Toy-nado makes organizing and visualizing toy data simple and efficient.

Made with: Java (JavaFX) CSS SQL (SQLite)

Election System Website

The Election System (made with 2 other friends) is a responsive, user-friendly platform designed for managing simple elections. It features secure functionalities such as real-time results, CAPTCHA protection, password encryption, SQL injection prevention, and session handling, with three user roles: administrator, organizer, and voter. Administrators and organizers manage the election setup and users, while voters can easily cast and view votes once registered or added to the system.

Made with: Java HTML CSS Javascript Bootstrap SQL

Gym Website (Front End)
